## Описание проекта

Этот репозиторий — портфолио специалиста технической поддержки и сопровождения информационных систем.

Здесь собраны реальные кейсы, процессы и инструкции, отражающие опыт работы с пользователями, IT-системами и бизнес-процессами.

## Навыки, которые демонстрируются

* Техническая поддержка пользователей (1–2 линия)
* Работа с инцидентами и заявками (Service Desk / ITSM)
* Администрирование учетных записей (Active Directory)
* Диагностика и решение пользовательских проблем
* Описание и оптимизация процессов поддержки
* Работа с бизнес-процессами и документами
* Работа с большими объемами данных (Excel)
* Подготовка инструкций и обучающих материалов

## Структура

* `cases/` — описание реальных кейсов из опыта работы
* `knowledge-base/` — инструкции и база знаний
* `diagrams/` — схемы процессов

## Цель проекта

Показать практический опыт в сопровождении IT-систем, поддержке пользователей и работе с бизнес-процессами.
