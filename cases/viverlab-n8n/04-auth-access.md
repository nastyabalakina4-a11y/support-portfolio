# 4. Авторизация и закрытие публичного доступа

**Статус:** 🔲 не начато

**Задача:** безопасный вход (логин + надёжный пароль), публичный доступ без авторизации закрыт.

## Пошагово
- [ ] Убедиться, что n8n User Management включён (по умолчанию включён — вход невозможен без создания аккаунта)
- [ ] Проверить, что пароль владельца (Owner) достаточно надёжный (буквы разного регистра, цифры, спецсимволы, 12+ символов)
- [ ] Убедиться, что без авторизации редактор n8n недоступен (открыть в приватном окне браузера без входа — должна запрашиваться авторизация)
- [ ] (опционально) Рассмотреть двухфакторную аутентификацию для аккаунтов n8n, если доступна в используемой версии

## Что нужно узнать
- [ ] Как n8n hashes/хранит пароли (bcrypt) — не критично для использования, но полезно для понимания
- [ ] Поддерживает ли текущая версия n8n 2FA в Community Edition
