# 2. Домен

**Статус:** ⛔ заблокировано — нужно купить домен

**Задача:** настроить доступ к n8n по отдельному домену/поддомену (например, `n8n.viverlab.ru` или `automation.viverlab.ru`).

## Пошагово
- [ ] Выбрать и зарегистрировать домен (например, через firstvds, там же где арендован сервер — проще с DNS в одном месте)
- [ ] Создать A-запись поддомена (например, `n8n.<домен>`) на IP сервера
- [ ] Дождаться распространения DNS (обычно от нескольких минут до пары часов)
- [ ] Проверить: `nslookup n8n.<домен>` должен возвращать IP сервера

## Что нужно узнать
- [ ] TTL записи и как ускорить распространение при необходимости
- [ ] Разница между доменом верхнего уровня и поддоменом для этой задачи (поддомен проще и обычно бесплатен в рамках уже купленного домена)
