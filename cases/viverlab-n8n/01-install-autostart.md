# 1. Установка n8n + автозапуск после перезагрузки

**Статус:** 🔲 не начато

**Задача:** актуальная стабильная версия n8n через Docker + Docker Compose, с автоматическим запуском после перезагрузки сервера.

## Пошагово
- [ ] Проверить, что установлена актуальная стабильная версия n8n (образ `docker.n8n.io/n8nio/n8n`, тег `latest` уже используется — n8n уже развёрнут на предыдущем этапе)
- [ ] Проверить, что сама служба Docker включена в автозагрузку системы (`systemctl is-enabled docker`)
- [ ] Убедиться, что контейнер n8n имеет `restart: unless-stopped` в `docker-compose.yml` (уже есть)
- [ ] Практически проверить автозапуск — перезагрузить сервер и убедиться, что n8n поднялся сам, без ручного вмешательства

## Что нужно узнать
- [ ] Разница между политиками restart в Docker: `no`, `on-failure`, `always`, `unless-stopped`
