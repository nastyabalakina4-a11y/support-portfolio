# 9. Планируемые автоматизации (справочно)

**Статус:** 🔲 не начато — список из ТЗ, к реализации после продакшен-настройки (пункты 1-7) и подключения [интеграций](08-integrations.md)

Процессы, которые планируется реализовать в виде workflow n8n:

- [ ] Автоматическое создание контента
- [ ] Перевод материалов на несколько языков
- [ ] Автопостинг в социальные сети
- [ ] Обработка обучающих материалов
- [ ] Генерация конспектов, чек-листов и FAQ
- [ ] Ведение базы знаний в Notion
- [ ] Автоматизация внутренних процессов команды
- [ ] Автоматизация онбординга сотрудников
- [ ] Автоматизация продаж и CRM
- [ ] Автоматизация клиентских проектов Viver Lab

## Интеллектуальный автопостинг (детализация из обновлённого ТЗ)
- [ ] Единый источник контента (Notion или Google Sheets): ID, текст, заголовок, язык, автор, площадки, дата, статус, вложения
- [ ] Публикация одного материала сразу в несколько площадок, с выбором площадок под каждую публикацию
- [ ] Автоматический перевод контента на нужные языки
- [ ] Адаптация текста под особенности площадки через ChatGPT/Claude — отдельные заголовки, описания, CTA
- [ ] Автооптимизация длины текста, хештегов и ключевых слов
- [ ] Публикация по расписанию с учётом часовых поясов, автовыбор времени по статистике
- [ ] Сбор ссылок и статистики по опубликованным материалам, анализ эффективности через ИИ
- [ ] Логирование: дата, площадка, статус, ссылка на пост, ID публикации, ошибки; экспорт в Google Sheets/Excel/CSV
- [ ] Архитектура должна позволять подключать новые платформы без переработки системы

## Требования к инфраструктуре (сквозные, из ТЗ)
- Стабильная работа 24/7
- Возможность масштабирования
- Простота дальнейшего администрирования
- Возможность подключения дополнительных сервисов без переустановки системы
