# Этап 7. Хостинг статичных сайтов (Nginx)

**Статус:** 🔲 не начато

**Задача:** использовать сервер как площадку для размещения статичных HTML-сайтов.

## Пошагово
- [ ] Установить Nginx: `apt install nginx`
- [ ] Разместить первый статичный HTML-сайт в каталоге раздачи
- [ ] Настроить виртуальные хосты под несколько сайтов (при необходимости)
- [ ] Порты 80/443 уже открыты в firewall (см. [этап 2](02-basic-hardening.md))

## Что нужно узнать
- [ ] Критерии, по которым можно оценить, потянет ли сервер конкретный сайт: тип сайта (статика/база данных/медиа), количество одновременных посетителей
- [ ] Инструменты мониторинга нагрузки на сервер: `htop`, `uptime` (load average), `free -h`
