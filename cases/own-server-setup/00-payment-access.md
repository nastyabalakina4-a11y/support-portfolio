# Этап 0. Сразу после оплаты

**Статус:** ✅ завершено

**Задача:** получить доступ к серверу и понять, как с ним работать через панель хостера.

## Что получите от хостера
- Письмо/уведомление в личном кабинете с IP-адресом сервера
- Логин (обычно `root`) и пароль — либо в письме, либо в разделе "Услуги" → ваш сервер в панели my.firstvds.ru

## Что нужно узнать заранее
- [x] Где в личном кабинете firstvds смотреть IP и root-пароль сервера (обычно раздел "Мои услуги" → карточка сервера)
- [ ] Как переустановить ОС через панель хостера, если что-то пойдёт не так (пригодится не раз на этапе обучения)

## Что сделано на практике
- Арендован VPS: 1 CPU, 2 ГБ RAM, 40 ГБ SSD, Ubuntu 24.04 LTS
- Получены IP и root-пароль через панель хостинга
- PTR-запись (reverse DNS) сервера по умолчанию содержала имя, идентифицирующее аккаунт — изменена через панель хостинга на нейтральное значение

## Результат
Доступ к серверу получен, базовые параметры (ОС, ресурсы, сеть) зафиксированы.
