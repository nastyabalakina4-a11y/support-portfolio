Настройка сервера для работы

VPS на Ubuntu 24.04, настроенный с нуля: доступ, защита, VPN, мониторинг, резервное копирование и хостинг — шаг за шагом, вручную, с пониманием каждой команды.

Ubuntu 24.04 Nginx Docker cron

🔍 — клик открывает схему

Базовая защита сервера

Вход по SSH-ключу вместо пароля, доступ под root закрыт, нестандартный порт SSH, firewall (ufw) и fail2ban против перебора паролей.

✓ Завершено

VPN на WireGuard

Собственный VPN-туннель до российского сервера: доступ к российским сервисам из-за границы через маршрутизацию и NAT.

✓ Завершено

Мониторинг (Zabbix)

Zabbix Server дома, агент на сервере: метрики CPU, памяти, диска и сети в реальном времени плюс триггеры на критичные события.

✓ Завершено

Резервное копирование

Ежедневный архив ключевых конфигов по cron, ротация старых копий и проверенное восстановление плюс копия за пределами сервера.

✓ Завершено

Веб-сервер (Nginx)

Хостинг статичных сайтов через отдельный virtual host — эта страница тоже раздаётся именно так.

✓ Завершено
