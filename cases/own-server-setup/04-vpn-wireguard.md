# Этап 4. Настройка VPN (WireGuard)

**Статус:** 🔲 не начато

**Задача:** получить доступ к российским сервисам из-за границы.

## Пошагово
- [ ] Установить WireGuard: `apt install wireguard`
- [ ] Сгенерировать ключи сервера и клиента (`wg genkey`, `wg pubkey`)
- [ ] Настроить конфиг сервера (`/etc/wireguard/wg0.conf`) — IP-адрес туннеля, порт, ключи
- [ ] Включить IP-forwarding в `/etc/sysctl.conf`
- [ ] Настроить NAT-правила (iptables/nftables), чтобы трафик клиента шёл в интернет через сервер
- [ ] Открыть нужный порт в `ufw`
- [ ] Создать конфиг для клиента (телефон/ноутбук) — можно сгенерировать QR-код для удобного подключения с телефона
- [ ] Проверить: подключиться с зарубежного устройства и зайти на сайт, который обычно недоступен

## Что нужно узнать
- [ ] Как работает split-tunneling (весь трафик через VPN vs только выборочные домены/IP)
- [ ] Как проверить, что ваш IP при подключении действительно российский (сайты типа 2ip.ru)
- [ ] Что делать, если провайдер за границей блокирует протокол WireGuard (запасной вариант — VLESS+Reality или Shadowsocks)
