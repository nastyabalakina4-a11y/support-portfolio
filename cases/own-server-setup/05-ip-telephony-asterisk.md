# Этап 5. IP-телефония (Asterisk + SIP-транк)

**Статус:** 🔲 не начато

**Задача:** звонить клиентам из-за границы с российским номером.

## Пошагово
- [ ] Зарегистрироваться у SIP-провайдера (например, Zadarma) и арендовать московский номер
- [ ] Установить Asterisk: `apt install asterisk` (или FreePBX для GUI-управления)
- [ ] Настроить подключение к SIP-транку провайдера (файл `pjsip.conf` или через веб-интерфейс FreePBX)
- [ ] Настроить исходящий диалплан — правила, как набранный номер уходит через транк
- [ ] Установить SIP-клиент на свой телефон/компьютер (например, Zoiper, MicroSIP) и подключить его к вашему Asterisk
- [ ] Открыть нужные порты для SIP/RTP в `ufw` (обычно 5060 и диапазон RTP-портов)
- [ ] Тестовый звонок — проверить, что у клиента отображается ваш российский номер

## Что нужно узнать
- [ ] Разница между SIP-сигнализацией и RTP (голосовым потоком) — важно для настройки firewall и NAT
- [ ] Как настроить NAT в Asterisk, если сервер и/или клиент за NAT (частая причина проблем со звуком)
- [ ] Тарифы SIP-провайдера на исходящие звонки по России и лимиты на количество одновременных линий
