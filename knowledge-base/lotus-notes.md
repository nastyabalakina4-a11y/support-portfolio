# Lotus Notes (корпоративная почта)

## Основные задачи:

- Работа с корпоративной почтовой системой Lotus Notes
- Отправка, получение и сортировка писем
- Обработка входящих запросов через почту
- Работа с вложениями (проверка, отправка, получение файлов)
- Поиск писем и информации в архиве
- Использование календаря для планирования встреч и задач

## Типовые ситуации:

- Пользователь не получает письма
- Проблемы с отправкой писем
- Ошибка доступа к почтовому ящику
- Потеря или поиск письма в архиве

## Базовая диагностика:

- Проверка корректности адреса получателя
- Проверка подключения к сети/VPN
- Проверка доступности почтового сервера
- Проверка заполненности почтового ящика
- Перезапуск клиента при сбоях

## Результат работы:

Обеспечение стабильной работы с корпоративной почтой и помощь пользователям в решении базовых проблем доступа и отправки сообщений.

# Lotus Notes (база договоров и корпоративные данные)

## Основные задачи:

- Работа с корпоративной системой Lotus Notes как с базой договоров
- Ведение и актуализация информации по договорам
- Внесение дополнительных соглашений в карточки договоров
- Прикрепление и открепление ЗЛ (зависимых лиц / связанных записей)
- Внесение изменений в существующие договоры
- Проверка корректности и актуальности данных в системе

## Типовые операции:

- Добавление дополнительных соглашений к договору
- Обновление информации по условиям договора
- Управление связанными записями (ЗЛ)
- Исправление или актуализация данных по запросу
- Поиск и проверка информации по договорам

## Типовые ситуации:

- Нужно добавить новое доп. соглашение к договору
- Изменились условия договора — требуется обновление записи
- Необходимо открепить или заменить ЗЛ
- Ошибка или несоответствие данных в карточке договора

## Базовая логика работы:

- Проверка актуальности существующей записи
- Внесение изменений согласно внутреннему запросу
- Контроль корректности прикреплённых документов и связей
- Сохранение целостности данных в системе

## Результат работы:

Актуализированная и корректная база договоров, отражающая все изменения, дополнительные соглашения и связанные записи.
