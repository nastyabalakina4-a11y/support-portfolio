# Active Directory (базовые операции)

## Основные задачи, с которыми я работала:

- Сброс пароля пользователя
- Разблокировка учетной записи
- Проверка прав доступа
- Назначение групп безопасности
- Диагностика проблем входа в систему

## Типовые проблемы:

- Пользователь не может войти в систему
- Заблокирована учетная запись
- Нет доступа к сетевым ресурсам

## Решение:

Проверка статуса учетной записи в Active Directory, анализ групп доступа, выполнение разблокировки или корректировки прав доступа.
